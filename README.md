# もやっとこねくと

もやっとしたアイディアを残して、あとからつなげるスマホ向けWebアプリのプロトタイプです。

## 目的

ワークショップや企画を考えるとき、思いつきを雑に残しつつ、後から使えそうなものだけを取り出して順番やまとまりを作れるようにするために作っています。

## 主な機能

- ideaタブでアイディアを追加
- アイディアを右スワイプして重要化
- connectタブで重要アイディアを整理
- アイディアをタップして編集・削除
- connectタブでアイディアをドラッグして並び替え
- アイディア同士の間にタイトル付きカラー付箋を追加
- タイトルをタップして編集
- ブラウザ内に自動保存

## 起動方法

```bash
npm install
npm run dev
```

表示されたローカルURLをブラウザで開きます。

## 技術

- React
- TypeScript
- Vite
- CSS
- localStorage
