# もやっとこねくと(仮名)


もやっとしたアイディアを残して、あとからつなげるスマホ向けWebアプリのプロトタイプです。

## 目的

企画を考えるとき、思いつきを雑に残しつつ、後から使えそうなものだけを取り出して順番やまとまりを作れるようにするために作っています。気軽に「こざね法」をスマホでもやりたかったので作りました。

## 主な機能

- ideaタブでアイディアを追加
- アイディアを右スワイプしてconnectタブに飛ばす
- connectタブで重要アイディアを整理
- アイディアをタップして編集・削除
- connectタブでアイディアをドラッグして並び替え
- アイディア同士の間にタイトル付きカラー付箋を追加
- ブラウザ内に自動保存


## 技術

- React
- TypeScript
- Vite
- CSS

## 状態

現在はプロトタイプ段階です。まず操作感を確認しながら、UIやデータ構造を改善していく予定です。
